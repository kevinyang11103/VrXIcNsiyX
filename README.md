## 项目概述

本项目是一款基于Python和深度学习技术的音乐推荐系统。该系统能够根据用户行为和偏好动态调整推荐内容，提供实时、个性化的音乐推荐服务。

- **后端语言与框架**：Python 3.x、Django

- **数据库**：MySQL
